Pip

Pip es la herramienta de gestión de paquetes de Python. Su nombre proviene de ""Pip Installs Packages"" (Pip Instala Paquetes). Pip se utiliza para instalar y gestionar paquetes de software escritos en Python que no forman parte de la biblioteca estándar de Python. Con pip, puedes instalar paquetes de Python desde el Python Package Index (PyPI), que es un repositorio en línea que contiene una amplia variedad de paquetes de software de Python creados y mantenidos por la comunidad de desarrolladores de Python.

Algunos de los comandos básicos de pip son:

Instalar un paquete:

pip install nombre_del_paquete

Desinstalar un paquete:

pip uninstall nombre_del_paquete

Listar paquetes instalados:

pip list

Actualizar un paquete:

pip install --upgrade nombre_del_paquete

Instalar paquetes desde un archivo de requisitos:

bash Copy code pip install -r requirements.txt Estos son solo algunos ejemplos de los comandos básicos de pip. La herramienta es muy versátil y te permite gestionar las dependencias de tu proyecto Python de manera fácil y eficiente.

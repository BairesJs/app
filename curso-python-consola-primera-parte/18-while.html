While



La estructura while se utiliza para ejecutar un bloque de código repetidamente mientras una condición específica sea verdadera. La sintaxis general es la siguiente: while condición: # Bloque de código a ejecutar mientras la condición sea verdadera Aquí tienes un ejemplo simple de un bucle while que cuenta hasta 5:



contador = 1

while contador <= 5:

print(contador)

contador += 1





En este caso, el bucle while se ejecutará mientras la variable contador sea menor o igual a 5. En cada iteración, imprime el valor de contador y luego incrementa su valor en 1. El bucle se detendrá cuando contador sea 6, ya que la condición contador <= 5 será falsa.

Ten en cuenta que es importante tener cuidado con las condiciones de salida de un bucle while para evitar bucles infinitos.

Es necesario asegurarse de que la condición eventualmente se vuelva falsa para que el bucle se detenga.

Puedes utilizar instrucciones de control, como break o return, para salir del bucle en ciertas condiciones.

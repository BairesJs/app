Atención

Manejo Básico del Sistema Operativo:

Aprender a moverte por el sistema operativo (Windows en tu caso), conocer la estructura de carpetas, saber cómo crear, copiar y mover archivos.

Instalación de Programas:

Aprender a instalar programas en tu sistema operativo. En particular, familiarizarte con la instalación de Python.

Copiar y Pegar:

Entender cómo copiar y pegar texto es esencial. Esto es útil al trabajar con código desde tutoriales, documentación o cualquier otro recurso en línea.

Editor de Texto vs. IDE:

Comprender la diferencia entre un editor de texto simple (como Notepad en Windows) y un Entorno de Desarrollo Integrado (IDE) más avanzado (como Visual Studio Code). Al principio, un editor de texto simple es suficiente.

Ejecutar un Programa Propio:

Entender qué significa ejecutar un programa que has escrito. Esto implica guardarlo con la extensión de archivo correcta (por ejemplo, .py para Python) y luego ejecutarlo desde la línea de comandos o terminal.

Línea de Comandos o Terminal:

Familiarizarte con la línea de comandos o la terminal en Windows (por ejemplo, PowerShell o el símbolo del sistema). Aprender comandos básicos como cd para cambiar de directorio, dir para listar archivos y python para ejecutar scripts Python.

Errores y Depuración:

Aprender a interpretar mensajes de error. Los errores son inevitables, pero entender lo que dicen los mensajes de error te ayudará a solucionar problemas en tu código.

Experimentar y Hacer Preguntas:

La programación es un proceso de aprendizaje continuo. Experimenta con el código, haz preguntas y busca respuestas en la documentación en línea y en foros de programación.
Recuerda que la programación es un conjunto de habilidades que se desarrollan con la práctica constante. No temas cometer errores, ya que son una parte integral del proceso de aprendizaje. ¡Diviértete programando!

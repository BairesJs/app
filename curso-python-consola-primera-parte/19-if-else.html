If Else


En Python, if y else son palabras clave utilizadas para controlar el flujo de un programa mediante la toma de decisiones condicionales.
La estructura básica es conocida como una declaración if-else.
Aquí hay un ejemplo simple:

edad = 18

if edad >= 18:

   print("Eres mayor de edad")

else:

   print("Eres menor de edad")



En este ejemplo, la condición edad >= 18 se evalúa. Si es verdadera, se ejecuta el bloque de código indentado después de if. Si es falsa, se ejecuta el bloque de código indentado después de else.
En este caso, si la variable edad es igual o mayor a 18, se imprime "Eres mayor de edad", de lo contrario, se imprime "Eres menor de edad".
También puedes tener múltiples condiciones usando elif (una abreviatura de "else if") para manejar varios casos:

puntuacion = 75

if puntuacion >= 90:

   print("A")

elif puntuacion >= 80:

   print("B")

elif puntuacion >= 70:

   print("C")

else:

   print("F")



En este ejemplo, se evalúan varias condiciones en orden. Si la puntuación es mayor o igual a 90, imprime "A".
Si no, verifica si es mayor o igual a 80, imprime "B", y así sucesivamente.
Si ninguna de las condiciones anteriores es verdadera, se ejecuta el bloque de código después de else y se imprime "F".
Tener en cuenta que hay momentos que debemos usar identacion para ordenar el codigo, leer el capítulo correspondiente.
